# onebyte

toggle csgo wallhack & radar with 1 byte patch.

## usage
1. build with vs (c++17, release x86)
2. run csgo.exe & onebyte.exe (order doesn't matter)
3. press `1` for wallhack, `2` for radar, `0` to exit

## notes
- only tested on steam csgo (legacy)
- no vac on legacy csgo, can't get banned
